# 💬 Real-Time Chat App

A real-time chat application built with Node.js and Socket.io.
Multiple users can chat simultaneously in their browser.

## 🛠️ Tech Stack
- Node.js
- Express
- Socket.io

## 🚀 How to Run

1. Clone the repo
   git clone https://github.com/your-username/chat-app.git

2. Install dependencies
   npm install

3. Start the server
   npm start

4. Open http://localhost:3000 in multiple tabs and start chatting!

## 💡 Features
- Real-time messaging
- Multiple users supported
- Username support
- Timestamps on messages
